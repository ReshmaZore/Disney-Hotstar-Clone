# Disney+ Hotstar Clone

This is a **frontend-based clone** of **Disney+ Hotstar**, built using **Vite, React, and Tailwind CSS**. It fetches movies and TV shows using **The Movie Database (TMDb) API** and is deployed on **Vercel**.
